Microsoft Azure AZ-900

AZ-900, also known as Microsoft Azure Fundamentals, is a foundational-level certification offered by Microsoft that focuses on providing a basic understanding of cloud concepts and services provided by the Microsoft Azure platform .

Py4E Certification

PY4E offers a comprehensive curriculum that covers fundamental programming concepts, data structures, web scraping, database integration, and web application development using Python.

Google Python Crash Cource

The Google Python Crash Course covers essential concepts and features of Python, including variables, data types, control flow, functions, modules, and file I/O. The course also introduces learners to object-oriented programming (OOP) in Python and covers topics such as classes, inheritance, and polymorphism.
